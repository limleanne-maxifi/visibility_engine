Proposed palette tokens

  • --navy #0f1c3a — page chrome (header band, footer, dark sections). Matches the marketing-site background.
  • --navy-2 #1a2744 — elevated dark panels (pipeline callout, CTA card). Already in use today.
  • --gold #D4A13F — primary accent. Replaces #534AB7 everywhere (buttons, focus rings, weights, links).
  • --gold-hover #b8862d — button hover state.
  • --gold-soft #fbf3df — soft gold for the score total row (replaces #EEEDFE).
  • --card #ffffff — form card stays white for readability and accessibility on long data-entry steps.

What changes in code

  • Find/replace #534AB7 → #D4A13F across components and emails (~30 occurrences).
  • Find/replace #6B5DD3 → #D4A13F (~10 occurrences) and #3C3489 → #b8862d.
  • Replace #EEEDFE backgrounds with #fbf3df.
  • MultiStepForm.tsx wrapper: add dark-navy header band behind the BrandPill.
  • BrandPill.tsx: white text on dark background instead of navy text on white.
  • Upsell card (results page Option A + email): swap purple background for navy with gold button.
  • Pipeline callout: tint the eyebrow text gold instead of blue.

Trade-offs to weigh

  • Dark band around form vs full dark form. This mockup keeps the form card light because data-entry forms with dark backgrounds tend to have weaker contrast on placeholders and validation states. If you want full dark-mode (Option A), the form fields would need a separate pass for input contrast and focus indicators.
  • Score total row. The current purple-tinted total row (#EEEDFE / #3C3489) translates to a warm-cream tint with navy text. Reads softer than the current purple; preserves "this is the conclusion" hierarchy.
  • Pipeline callout. Already navy in production — only the small eyebrow accent colour changes (blue-300 → gold). Minimal disruption, high brand-cue payoff.
